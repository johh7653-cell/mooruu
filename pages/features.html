Core Features

The product is not just a chat window. It is a persistent AI character with social rules around how it grows.

  • 01
    On-chain memory

    Important memories, versions, and milestones can be recorded so the character has a traceable history.

  • 02
    Community governance

    Major personality changes are proposed, debated, voted, and executed by the community.

  • 03
    Treasury runtime

    The token acts like electricity. The treasury pays to keep the AI alive and growing.

  • 04
    Model portability

    If one model stops serving the character, the community can connect a new one without killing the soul.
